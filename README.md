# BlueCollar Systems Website

## Tech Stack

- Vanilla HTML5
- [Tailwind CSS](https://tailwindcss.com) (CDN)
- Zero build dependencies

## Development

Open `index.html` in a browser. No build step required.

## Deployment

Deployed via Cloudflare Pages on push to `main`.
