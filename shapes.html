Checksum Verification

Verify your download before use. The hash output should match the SHA-256 value listed above.

Windows (PowerShell)

Run this from the folder containing the ZIP file:

Get-FileHash .\Structural-Steel-SU-Shapes-latest.zip -Algorithm SHA256

Get-FileHash .\Structural-Steel-DXF-DWG-Shapes-latest.zip -Algorithm SHA256

macOS / Linux

Run from the folder containing the ZIP file:

shasum -a 256 Structural-Steel-SU-Shapes-latest.zip

shasum -a 256 Structural-Steel-DXF-DWG-Shapes-latest.zip
